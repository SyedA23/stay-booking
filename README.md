## Overview

This project is a responsive stay booking web application built as part of a React machine test.

## Features

- Modern responsive UI
- React Hooks (useState, useEffect, useRef)
- Custom hook for API calls
- Error handling and retry
- Skeleton loading UI
- Filtering and sorting
- Dark mode toggle
- Debounced search
- Friendly error and empty states

## Tech Stack

React, Vite, Tailwind CSS

## How to Run

1. Clone the repo
2. npm install
3. npm run dev
